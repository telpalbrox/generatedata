## UUID v4 Data Type

This Data Type generates a random UUID v4 of the form: *xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx*  


### Example API Usage

```javascript
{
    "numRows": 20,
    "rows": [
        {
            "type": "UUID",
            "title": "UUID"
        }
    ],
    "export": {
        "type": "JSON",
        "settings": {
            "stripWhitespace": false,
            "dataStructureFormat": "complex"
        }
    }
}
```
